## 项目组件位置总览
### 1. PPT 渲染组件
位置： components/slide-renderer/Editor/index.tsx

这个是幻灯片编辑器的入口点，根据模式（autonomous/playback）选择显示不同的画布组件。

### 2. 元素渲染组件
位置： components/slide-renderer/components/element/ 各个元素的渲染器：

- TextElement/index.tsx - 文本元素
- ImageElement/index.tsx - 图片元素
- ShapeElement/index.tsx - 形状元素
- LatexElement/index.tsx - LaTeX 公式
- ChartElement/index.tsx - 图表
- TableElement/index.tsx - 表格
- VideoElement/index.tsx - 视频
- LineElement/index.tsx - 线条
### 3. 视觉特效实现 聚光灯效果
位置： components/slide-renderer/Editor/SpotlightOverlay.tsx

这个组件实现了聚光灯动画效果，通过 SVG mask 实现高亮区域，周围变暗，有平滑的淡入淡出和缩放动画。
 激光笔效果
位置： components/slide-renderer/Editor/LaserOverlay.tsx

激光笔特效实现，从边角平滑飞入目标位置，带脉冲发光效果。

### 4. 白板渲染组件
位置： components/whiteboard/

- index.tsx - 白板主组件，含 UI 控件
- whiteboard-canvas.tsx - 画布实现
- whiteboard-history.tsx - 历史记录
### 5. 播放控制 UI
位置： components/canvas/

- canvas-area.tsx - 画布区域，含播放按钮提示
- canvas-toolbar.tsx - 工具栏，含播放/暂停、翻页、全屏等控件
### 6. 核心播放引擎
位置： components/stage.tsx

这个是整个舞台的主组件，集成了 PlaybackEngine 、 ActionEngine ，管理播放状态、场景切换、多智能体互动等功能。

### 7. 场景渲染器
位置： components/stage/scene-renderer.tsx

根据场景类型（slide/quiz/interactive/pbl）选择对应的渲染组件。

### 8. 场景元素显示
位置： components/slide-renderer/Editor/ScreenElement.tsx

场景元素的统一入口，根据元素类型选择对应的渲染组件。

### 9. 屏幕画布
位置： components/slide-renderer/Editor/ScreenCanvas.tsx

集成了所有视觉效果（聚光灯、激光笔、缩放）的画布组件。

所有这些组件通过 Zustand 状态管理（ lib/store/canvas.ts ）和 SceneContext 上下文进行通信。
