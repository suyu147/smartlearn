### 2.1 `quiz-player.tsx` — 题目练习

| 项目 | 说明 |
|------|------|
| **来源** | [components/scene-renderers/quiz-view.tsx](file:///D:/python/docment/OpenMAIC-main/components/scene-renderers/quiz-view.tsx) |
| **复用方式** | 适配修改 |
| **原因** | 核心答题/批改逻辑通用，但类型定义和 Store 依赖需替换 |

#### 需要修改的内容

| 修改项 | 原代码 | 改为 |
|--------|--------|------|
| 类型导入 | `import { QuizQuestion } from '@/lib/types/stage'` | `import { Quiz } from '@/lib/types/resource'`（新类型定义） |
| 草稿缓存 key | `sceneId` 作为 key | `quizId` 作为 key |
| 模型配置 | `getCurrentModelConfig()` | `useSettingsStore(s => s.modelId)` |
| 批改 API | `/api/quiz-grade` | `/api/generate/quiz/grade` |
| Props | `{ questions: QuizQuestion[], sceneId: string }` | `{ quiz: Quiz, onComplete?: (result: QuizResult) => void }` |

#### 保留的核心逻辑

- 四阶段状态机：`not_started → answering → grading → reviewing`
- 单选/多选/简答题渲染
- 选择题本地批改 + 简答题 AI 批改
- 草稿缓存（`useDraftCache`）
- 语音输入（`SpeechButton`）
- 得分环形图 + 回顾页

#### 适配提示词

```
请基于以下 OpenMAIC 源文件进行适配改造：

源文件：components/scene-renderers/quiz-view.tsx
目标文件：components/resources/quiz-player.tsx

改造要求：
1. 将 Props 从 { questions: QuizQuestion[], sceneId: string } 改为 { quiz: Quiz, onComplete?: (result: QuizResult) => void }
2. Quiz 类型定义如下：
   interface Quiz {
     id: string;
     type: 'choice' | 'fill' | 'short_answer' | 'coding' | 'case_analysis';
     difficulty: 1|2|3|4|5;
     knowledgePoints: string[];
     question: string;
     options?: string[];
     answer: string | string[];
     explanation: string;
     hints: string[];
     autoGradable: boolean;
   }
3. 移除 getCurrentModelConfig() 调用，改为 useSettingsStore(s => s.modelId)
4. 草稿缓存 key 从 sceneId 改为 quiz.id
5. 批改 API 从 /api/quiz-grade 改为 /api/generate/quiz/grade
6. 保留四阶段状态机、题目渲染、批改逻辑、得分展示、语音输入等核心功能
7. 保留所有动画效果（motion/react）和 i18n 支持
8. 新增编程题类型渲染：使用代码编辑器（Monaco Editor / CodeMirror）+ 测试用例展示
9. 新增案例分析题渲染：展示案例场景 + 开放式问答
10. onComplete 回调在 reviewing 阶段触发，传递 QuizResult { quizId, score, answers, timeSpent }
```

### 3.1 `profile-chat.tsx` — 画像构建对话

| 项目 | 说明 |
|------|------|
| **参考来源** | [components/scene-renderers/pbl/chat-panel.tsx](file:///D:/python/docment/OpenMAIC-main/components/scene-renderers/pbl/chat-panel.tsx) 的对话布局 + [components/ai-elements/conversation.tsx](file:///D:/python/docment/OpenMAIC-main/components/ai-elements/conversation.tsx) 的自动吸底 |
| **核心功能** | 多轮对话抽取学生画像特征，实时展示画像更新 |

#### 提示词

```
请创建一个对话式学习画像构建组件。

文件路径：components/profile/profile-chat.tsx

功能描述：
通过 AI Agent 多轮对话，自动抽取学生的学习特征，构建包含 8 个维度的学习画像。

Props 接口：
interface ProfileChatProps {
  userId: string;
  onProfileUpdate: (dimensions: Partial<LearningProfileDimensions>) => void;
  onComplete: () => void;
  initialProfile?: LearningProfileDimensions | null;
}

LearningProfileDimensions 包含 8 个维度：
- knowledgeBase: { level: 'beginner'|'intermediate'|'advanced'; subjects: { name: string; mastery: number }[] }
- cognitiveStyle: { type: 'visual'|'auditory'|'reading'|'kinesthetic'; preference: string }
- learningGoals: { shortTerm: string[]; longTerm: string; targetExam?: string }
- weakPoints: { topics: string[]; errorPatterns: string[] }
- timePreference: { preferredDuration: number; preferredTimeSlot: string; frequency: 'daily'|'weekly'|'irregular' }
- interests: { domains: string[]; preferredFormats: ('document'|'video'|'code'|'quiz'|'mindmap')[] }
- learningPace: { speed: 'slow'|'moderate'|'fast'; depthPreference: 'broad'|'deep' }
- errorPatterns: { commonMistakes: string[]; difficultAreas: string[] }

UI 要求：
1. 左侧：对话区域
   - 使用 Conversation + ConversationContent 组件（from @/components/ai-elements/conversation）实现自动吸底滚动
   - 用户消息右对齐，Agent 消息左对齐（使用 AgentAvatar 显示 Agent 头像）
   - Agent 消息使用 MessageResponse（from @/components/ai-elements/message）渲染 Markdown
   - 流式输出时显示 shimmer 加载效果
   - 输入框在底部，支持语音输入（SpeechButton from @/components/audio/speech-button）
   - 使用 useDraftCache 缓存输入内容
2. 右侧：实时画像预览面板
   - 显示当前已抽取的画像维度（已填充的显示绿色勾，未填充的显示灰色）
   - 每个维度以简洁卡片形式展示关键信息
   - 画像更新时有高亮动画效果
3. 对话流程：
   - Agent 首先打招呼并引导对话
   - 每轮对话后，Agent 尝试从回复中抽取画像特征
   - 抽取结果通过 onProfileUpdate 实时回调
   - 所有维度填充完成后，显示"画像构建完成"按钮

SSE 通信：
- POST /api/profile/chat 发送消息
- SSE 事件类型：agent_start, text_delta, profile_update, agent_end
- 使用 StreamBuffer（from @/lib/buffer/stream-buffer）处理流式文本
- 使用 processSSEStream（from @/lib/chat/process-sse-stream）解析 SSE

技术栈：React 19 + TypeScript + Tailwind CSS 4 + shadcn/ui + motion/react + lucide-react
依赖组件：Conversation, ConversationContent, ConversationScrollButton, MessageResponse, AgentAvatar, SpeechButton, StreamBuffer
```
