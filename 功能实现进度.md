## 实现详情

### #2 思维导图Markmap渲染 ✅
- 重写 `components/resources/mindmap-viewer.tsx`
- 使用 `markmap-view` 的 `Markmap` 类 + `markmap-lib` 的 `Transformer` 渲染交互式思维导图
- 支持缩放、拖拽、折叠交互
- 添加加载状态和错误处理（fallback 到纯文本）
- 新增依赖：`markmap-lib`

### #11 文档Markdown渲染 ✅
- 重写 `components/resources/document-viewer.tsx`
- 使用 `react-markdown` + `remark-gfm` 渲染 Markdown
- 代码块复用项目已有的 `CodeBlock` 组件（基于 shiki 语法高亮）
- 添加 `MarkdownErrorBoundary` 错误边界
- 新增依赖：`react-markdown`、`remark-gfm`

### #10 Quiz自动评分 ✅
- 重写 `components/resources/quiz-player.tsx`
- 添加 `checkAnswer` 答案对比逻辑（支持选择题和填空题）
- 评分结果面板：得分、正确题数/总题数、进度条
- 每题正确/错误状态标记（✓/✗）
- 选择题正确答案绿色高亮，错误选项红色标记
- 填空题文本输入框，提交后显示对错
- 答错时显示正确答案

### #4 Tavily搜索实现 ✅
- 重写 `lib/web-search/tavily.ts`
- 实现 `searchWithTavily` 函数，调用 Tavily API
- 添加错误处理（HTTP 错误、网络错误）
- 添加响应时间计算
- `context` 字段格式化为带编号的文本

### #13 设置页配置传递到API ✅
- 修改4个API路由（profile/chat、generate/resources、path/plan、tutor/chat）
- 将硬编码的 `process.env` 模型配置替换为 `resolveModel` 函数
- 从请求体接收 `aiConfig`（providerId、modelId、apiKey、baseUrl）
- 修改4个前端调用方（profile-chat.tsx、resources/page.tsx、learning-path/page.tsx、tutor/page.tsx）
- 前端从 `useSettingsStore` 读取配置，通过 `aiConfig` 字段传递给 API
- `resolveModel` 优先使用客户端配置，未提供时回退到环境变量

### #3 代码在线运行 ✅
- 创建 `app/api/code/execute/route.ts`，使用 Piston API 执行代码
- 支持8种语言（Python、JavaScript、TypeScript、Java、C++、C、Go、Rust）
- 重写 `components/resources/code-runner.tsx`
- 添加语言选择下拉框、运行按钮、stdin输入、输出展示区域
- 自动语言检测、运行状态显示、错误处理

### #9 路径节点与资源关联 ✅
- 修改 `app/api/path/plan/route.ts`，从请求体接收 `resources` 参数
- 调用 `buildResourcePrompt` 将资源信息注入 LLM prompt
- 修改 `app/learning-path/page.tsx`，从 resources store 获取已有资源并传递给 API
- 修改 `components/learning-path/path-timeline.tsx`，在节点卡片中显示关联的资源链接和练习测验标签

### #1 视频生成API接入 ✅
